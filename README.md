# Shopify JSON Inspector

**Shopify JSON Inspector** 是一个纯前端工具，用于可视化分析 Shopify OS2.0 的 Page / Template JSON 文件，无需任何后端支持。  

---

## 核心功能

- **快速解析 JSON**：上传 Shopify Page 或 Template JSON，自动提取内容。  
- **Images Inspector**：列出所有 `shopify://shop_images/` 引用，支持一键打包下载。  
- **Sections Inspector**：显示页面所有 section 类型，并统计各类型出现次数。  
- **极简 UI**：以 Inspector 风格呈现，长列表也可流畅滚动，方便开发者快速定位信息。  
- **完全前端**：可部署在 GitHub Pages，无需 Node 或服务器环境。  

---

## 使用场景

- Shopify 店铺迁移时快速收集图片资源  
- 分析 OS2.0 模板结构  
- 调试或优化页面布局  
- 教学或工具演示用途  

---

## 快速开始

1. 打开页面，选择你的 Shopify JSON 文件  
2. （可选）输入 CDN 前缀，用于下载图片  
3. 浏览 Images / Sections 列表  
4. 点击「Download ZIP」下载所有图片
